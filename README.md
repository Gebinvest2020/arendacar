# Arendacar — сайт аренды автомобилей

Сайт для аренды (проката) автомобилей. Проект на ранней стадии: сейчас это чистый каркас на Next.js без базы данных, авторизации, оплаты и API бронирования — они будут добавлены позже.

## Стек

- **Next.js 16** (App Router)
- **React 19**
- **TypeScript**
- **Tailwind CSS**
- **ESLint**
- Структура с папкой `src`
- Алиас импортов `@/*`

## Требования

- Node.js 18.18+ (проект разрабатывался на Node.js 24)
- npm

## Как запустить локально

```bash
# 1. Установить зависимости (если ещё не установлены)
npm install

# 2. Запустить сервер разработки
npm run dev
```

После запуска откройте в браузере: [http://localhost:3000](http://localhost:3000)

## Полезные команды

```bash
npm run dev     # режим разработки
npm run build   # production-сборка
npm run start   # запуск production-сборки
npm run lint    # проверка кода ESLint
```

## Структура проекта

```
src/
  app/
    layout.tsx     # корневой layout
    page.tsx       # главная страница
    globals.css    # глобальные стили (Tailwind)
public/            # статические файлы
```
